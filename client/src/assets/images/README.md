# Image Assets for Hospital Appointment Booking System

## How to Replace Images

Simply replace the image files in this folder with your own images. Keep the same filenames.

### Required Images:

1. **logo.png** (Recommended: 200x60px, transparent background)
   - Used across all pages (login, dashboard, navigation)
   - Should be your hospital/clinic logo

2. **dashboard-hero.jpg** (Recommended: 1920x600px)
   - Main banner image on patient dashboard
   - Suggestion: Professional medical facility photo

3. **doctor-consultation.jpg** (Recommended: 800x600px)
   - Displayed in appointment booking section
   - Suggestion: Doctor consulting with patient

4. **medical-team.jpg** (Recommended: 800x600px)
   - Displayed in services/features section
   - Suggestion: Medical team or hospital staff

### Image Guidelines:
- Use high-quality images (at least 1200px width for banners)
- Keep file sizes optimized (under 500KB per image)
- Supported formats: JPG, PNG, WebP
- Use professional medical imagery

### Quick Replacement:
1. Navigate to: `frontend/src/assets/images/`
2. Replace the image file with the same name
3. Refresh your browser (Ctrl+F5)
